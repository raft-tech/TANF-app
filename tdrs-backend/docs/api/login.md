 
# Authentication Login via Login.gov
Supports registering/logging in a user via Login.gov and creates an httpOnly cookie for a secured [authentication](api/authentication.md).

---
**Request**:

`Get` `v1/login/oidc`

Parameters:

None

*Note:*

- Not Authorization Protected

**Response**:

```json
Content-Type application/json
200 Ok

{
  "user_id":"1234567-1235-123b-b88d-069792f4ca7b",
  "email":"tester@mailnator.com",
  "status":"New User Created!"
 }
```

This will return with an httpOnly cookie and be stored client side for
authenticating future requests to the API by providing in the header of API calls and a JSON response body composed of the following:

- **user_id**: String value randomly generated by the backend for each new user.
- **email**: A string value representing email address associated with their Login.gov account and registration with the backend services. 
- **status**: String value reflecting the operation performed.
---
**Request**:

`Get` `v1/logout/oidc`

Parameters:

None

*Note:*

- Not Authorization Protected

**Response**:

```json
Content-Type application/json
200 Ok

{
  "user_id":"1234567-1235-123b-b88d-069792f4ca7b",
  "email":"tester@mailnator.com",
  "status":"New User Created!"
 }
```

This will return with an httpOnly cookie and be stored by the client for
authenticating future requests to the API. See [Authentication](api/authentication.md).

----
**Failure to get valid response from Login.gov/token:**


```json
Content-Type application/json
403 Forbidden
{
   "error": ("Invalid Validation Code Or OpenID Connect Authenticator Down!")
}
```

----
**Failure to get valid response from Login.gov/token:**


```json
Content-Type application/json
404 Bad Request
{
   "error": "Invalid Validation Code Or OpenID Connect Authenticator Down!"
}
```
----
**Failure to get valid email from decoded token:**

```json
Content-Type application/json
404 Bad Request
{
   {"error": "Unverified email!"}
}

```
----
**Failure look-up/register user in database:**

```json
Content-Type application/json
404 Bad Request
{
   "error": "Email verfied, but experienced internal issue with login/registration."
}

```
